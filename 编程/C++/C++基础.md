# 简介
## Object based & Object Oriented
1. object based：分为不包括指针成员和包括指针成员两类
2. object oriented：学习classes之间的联系：继承、复合、委托。
## 头文件的防御式声明
```
#ifndef __COMPLEX__
#define __COMPLEX__
...
#endif
```

## inline函数
函数若定义在类内部，则自动成为inline函数

## 构造函数
冒号后面的称为：initialization list（初值列），作用是初始化
如果放到大括号中，则为：assignments（赋值）。

## 常量成员函数
当一个成员函数不会改变成员变量时，应该设置为const
```
double real() const {return re;}
```
real()函数返回成员变量，不应该改变成员变量的值。所以应该加上const。否则，当一个常量对象调用real()函数时就会出错。
常量成员函数可以被常量对象调用，也可以被非常量对象调用；
非常量成员函数只能被非常量对象调用。

## 返回值：引用或值
当一个变量是在函数内部创建的，那么函数的返回值类型不能是引用。因为函数内部创建的变量在函数结束时，就消失了，把一个不存在的东西当做引用传出去是错误的，这时候函数应该返回一个值，当返回的是一个值的时候，会把函数内创建的变量拷贝一份。
