# 1\. 反转单链表，返回反转后链表的头结点

1->2->3->4 => 4->3->2->1

```
ListNode* reverse(ListNode* head){
    if(!head || !head->next){ //当前结点为null或者其后继为null
        return head;
    }
    
    // 把当前结点后面的全部反转。这个newHead在递归时都是直接传出的，
    // 没有改变，所以一直都是原链表的最后一个结点。
    ListNode* newHead = reverse(head->next);
    
    // 反转当前结点和下个结点的顺序
    head->next->next = head; // 让下个结点的后继指针指向当前结点
    head->next = nullptr; // 当前结点的后继指针置空，目的是为了保证当
                          // 全部反转后，最后一个结点的后继指针为空
    return newHead;
} 
```

# 2\. 反转单链表的前K个结点。（头结点的序号为1）

```
ListNode* remain;
ListNode* reverseK(ListNode* head, int k){
    if(k == 1){
        remain = head->next; // 记录不需要反转的部分
        return head;
    }
    ListNode* newHead = reverseK(head->next, k - 1);
    head->next->next = head;
    head->next = remain; // 每个节点反转后，其后继指针都置为remain
                         // 当k个结点反转完成，最后一个结点的后继正
                         						// 好指向remain。中间的结点虽然也置为                            // remain，但是中间都会被head->next->next = head给修改。
    return newHead;
}
```

# 3\. 反转单链表中两个数\[left, right\]之间的结点

```
ListNode* reverseBetween(ListNode* head, int left, int right){
    if(left == 1){
        return reverseK(head, right);
    }
    head->next = reverseBetween(head->next, left-1, right-1);
    return head;
}
```

# 4\. k个一组反转单链表

你链表的头节点 head ，每 k 个节点一组进行翻转，请你返回修改后的链表。k 是一个正整数，它的值小于或等于链表的长度。如果节点总数不是 k 的整数倍，那么请将最后剩余的节点保持原有顺序。
**题解一：**
k个一组反转，相当于反转\[1, k\], \[k+1, 2k\], \[2k+1, 3k\] ...
相当于反转前k个结点，记录下remain结点；
前k个结点反转完成后，把remain结点作为剩下结点的头结点，继续反转k个结点。。。
这样就体现了一种递归的形式。

```
ListNode* remain;
ListNode* reverseK(ListNode* head, int k){
    if(k == 1){
        remain = head->next;
        return head;
    }
    ListNode* newHead = reverse(head->next, k-1);
    head->next->next = head;
    head->next = remain;
    return newHead;
}
ListNode* reverseKGroup(ListNode* head, int len, int k){
    if(len < k){
        return head;
    }
    ListNode* newHead = reverseK(head, k);
    ListNode* midHead = reverseKGroup(remain, len - k, k);
    head->next = midHead;
    return newHead;
}
ListNode* reverseKGroup(ListNode* head, int k){
    int len = 0;
    ListNode* node = head;
    while(node){
        len++;
        node = node->next;
    }
    ListNode* newHead = reverseKGroup(head, len, k);
    return newHead;
}
```

**题解二：**
把子问题看成是反转从头结点到某个结点之间的部分

```
// 反转整个单链表可以看成是反转【head, nullptr）之间的部分，注意是左
// 闭右开，那么反转【head, node）可以写成：
ListNode* reverse(ListNode* head, ListNode* node){
    if(head->next == node){
        return head;
    }
    ListNode* newHead = reverse(head->next, node);
    head->next->next = head;
    head->next = node;
    return newHead;
}

ListNode* reverseKGroup(ListNode* head, int k){
    if(!head)
        return nullptr;
    ListNode* node = head;
    for(int i = 0; i < k; i++){
        if(!node)
            return head;
        node = node->next;
    }
    
    ListNode* newHead = reverse(head, node);
    head->next = reverseKGroup(node, k);
    return newHead;
}
```
