## 解法：
1. 中心扩散法：
注意最后返回的时候，max_left需要+1。因为第一个while语句会导致left多移动一位，但是第二个while不会使right多移动。
```c++
		#include <iostream>
		#include <string>
		class Solution{
		public:
			std::string longestPalindrome(std::string s) {
				int max_left = 0, max_right = -1;
				for (int i = 0; i < s.size(); ++i) {
					int left = i, right = i;
					while(left >= 0 && s[left] == s[i]) --left;
					while(right <= s.size() -1 && s[right] == s[i]) right++;
					while(left >= 0 && right <= s.size()-1 && s[left] == s[right]) {
						left--;
						right++;
					}
					if (max_right - max_left < right - left) {
						max_left = left;
						max_right = right;
					}
				}
				return s.substr(max_left + 1, max_right - max_left -1);
			}
		};
```
