# 1\. 基本01背包

## 问题描述：

有N件物品和一个容量为C的背包；每件物品有且只有一件。如何选择物品，使得在不超过背包容量的条件下，获取的价值最大。

## 1.1 状态定义

dp\[i\]\[j\]：i 表示当前枚举到的物品，j 表示现在剩余容量，dp\[i\]\[j\]表示最大价值
即**考虑前i 件物品，使容量不超过C的条件下的背包的最大价值。**

## 1.2 状态转移方程

对于第i件物品，有**选** 和 **不选** 两种策略：
【不选】：等效于只考虑前i-1件物品，当容量为 j 时的最大价值：dp\[i-1\]\[j\]
【选】：则第 i 件物品会消耗v\[i\]的容量，获得w\[i\]的价值，那么留给前 i-1 个物品的容量为 j - v\[i\]。此时最大价值为：dp\[i-1\]\[j - v\[i\]\] + w\[i\]。
最后从【选】和【不选】中取价值更大的那个
则**状态转移方程为**：

```
dp[i][j] = max(dp[i-1][j], dp[i-1][j-v[i]] + w[i])
```

## 1.3 代码

### 常规解法 dp\[N\]\[C+1\]

```
int maxValue(int N, int C, vector<int>& v, vector<int>& w){
    vector<int> dp(N, vector<int>(C+1, 0));
    // 先处理考虑第一件物品的情况
    for(int j=0; j <= C; j++)
        dp[0][j] = j >= v[0] ? v[0] : 0;
    
    // 处理考虑其余物品的情况
    for(int i = 1; i < N; i++){
        for(int j=0; j <= C; j++){
            int no = dp[i-1][j];
            int yes = j >= v[i] ? dp[i-1][j-v[i]] + w[i] : 0;
            dp[i][j] = max(no, yes);
        }
    }
    return dp.back().back();
}
```

时间复杂度：有N*C个状态需要转移，复杂度为O(N*C);
空间复杂度：O(N*C);

### 优化1：滚动数组 dp\[2\]\[C+1\]

转移方程中，计算第i行时，只用到了第i-1行中的某些值，所以可以用只有两行的数组保存中间结果，计算的结果轮流往这两行中存储

```
int maxValue(int N, int C, vector<int>& v, vector<int>& w){
    vector<vector<int>> dp[2][C+1];
    for(int i = 0; i <= C; i++)
        dp[0][i] = i >= v[0] ? w[0] : 0;
        
    for(int i = 1; i < N; i++){
        for(int j = 0; j <=C; j++){
            int no = dp[(i-1)%2][j];
            int yes = j >= v[i] ? dp[(i-1)%2][j-v[i]] + w[i] : 0;
            dp[i%2][j] = max(no, yes);
        }
    }
    return dp.back().back();
}
```

时间复杂度不变
空间复杂度：O(C)

### 优化二：一维空间优化 dp\[i\]

从转移方程：dp\[i\]\[j\] = max(dp\[i-1\]\[j\], dp\[i-1\]\[j-v\[i\]\] + w\[i\]) 可以看出，第i行的结果不只是只依赖第i-1行，还只依赖第i-1行中的第j的值和第j-v\[i\]的值（对应不选和选两种）
为了确保计算dp\[j\]时，它前面的结果还是相对i-1的结果，所以需要容量从大到小遍历

```
int maxValue(int N, int C, vector<int>& v, vector<int>& w){
    vector<int> dp(C+1);
    for(int i = 0; i < N; i++){
        for(int j = C; j >= v[i]; j--){
            int no = dp[j];
            int yes = dp[j - v[i]] + w[i];
            dp[j] = max(no, yes);
        }
    }
    return dp[C];
}
```

# 2\. 完全01背包

## 2.1 问题描述：

基本01背包问题限制了每件物品只能选择一次，完全01背包问题中，每件物品的选择没有限制。

## 2.2 状态定义与基本01背包一样

## 2.3 状态转移方程

因为每个物品可以被选择多次，所以对于某个dp\[i\]\[j\]来说，其值应该是下列方案中的最大值：

- 选择0件物品 i 的最大价值：dp\[i-1\]\[j\]
- 选择1件物品 i 的最大价值：dp\[i-1\]\[j - v\[i\]\] + w\[i\]
- 选择2件物品 i 的最大价值：dp\[i-1\]\[j - 2 * v\[i\]\] + 2 * w\[i\]
- ...
- 选择k件物品 i 的最大价值：dp\[i-1\]\[j - k * v\[i\]\] + k * w\[i\]
    **常规状态转移方程**：

```
dp[i][j] = max(dp[i-1][j], dp[i-1][j - k*v[i]] + k*w[i]), 0<k*v[i] <=j
```

### 常规代码

```
int maxValue(int N, int C, vector<int>& v, vector<int>& w){
    vector<vector<int>> dp(N, vector<int>(C+1));
    for(int j = 0; j <= C; j++){
        int maxK = j / v[0];
        dp[0][j] = maxK * w[0];
    }
    
    for(int i = 1; i < N; i++){
        for(int j = 0; j <= C; j++){
            int no = dp[i-1][j];
            int yes = 0;
            for(int k = 1;; k++){
                if(j < v[i] * k)break;
                yes = max(yes, dp[i-1][j - k*v[i]] + k*w[i]);
            }
            dp[i][j] = max(no, yes);
        }
    }
    return dp.back().back();
}
```

时间复杂度：O(N*C*C)
空间复杂度：O(NC)

### 一维空间优化

对常规状态转移方程的k进行展开：

```
dp[i][j] = max(dp[i-1][j], { dp[i-1][j-v[i]]+w[i], dp[i-1][j-2*v[i]]+2*w[i], ..., dp[i-1][j-k*v[i]]+k*w[i] } ), 0<k*v[i]<=j
```

```
dp[i][j-v[i]] = max({ dp[i-1][j-v[i]], dp[i-1][j-2*v[i]]+w[i], dp[i-1][j-3*v[i]]+2*w[i], ..., dp[i-1][j-k*v[i]]+(k-1)*w[i] } ), 0<k*v[i]<=j
```

上面两个式子{ } 中的部分相差 w\[i\]，所以得出状态转移方程：

```
dp[i][j] = max(dp[i-1][j], dp[i][j-v[i]] + w[i])
```

然后消除 i 的维度，得到**状态转移方程**：

```
dp[j] = max(dp[j], dp[j-v[i]] + w[i])
```

在计算dp\[i\]\[j\]时，需要提前知道dp\[i\]\[j-v\[i\]\]，所以对 j 的遍历应该**从小到大**。
代码

```
int maxValue(int N, int C, vector<int>& v, vector<int>& w){
    vector<int> dp(C+1);
    for(int i = 0; i < N; i++){
        // for(int j = 0; j <= C; j++){
        for(int j = v[i]; j <= C; j++){
            int no = dp[j];
            // int yes = j >= v[i] ? dp[j-v[i]] + w[i] : 0;
            int yes = dp[j-v[i]] + w[i];
            dp[j] = max(no, yes);
        }
    }
    return dp.back();
}
```
从上面代码来理解：考虑每一件物品在不同背包容量下，所能达到的最大价值
比如：考虑第一件物品，在背包容量【1，2，3，4，5，C】下，分别最大价值为【v1, v2, v3, v4, v5, vC】。
考虑第二件物品时，在背包容量【1，2，3，4，5，C】下，计算最大价值时，其实是在更新【v1, v2, v3, v4, v5, vC】这些值。比如对于容量为4时，如果不选择第二件物品，那么此时最大价值就是v4；如果选择第二件物品，此时的最大价值是容量4减去一个第二件物品的体积的最大价值+第二件物品价值，假设容量4减去第二件物品体积之后的容量是2，在之前容量2的最大价值已经是考虑了第二件物品的情况下的最大价值，里面已经包含了0个或多个第二件物品了，所以对于容量4来说，也是已经考虑了多个第二件物品的最大价值了。

# 3\. 多重背包

## 3.1 问题描述

多重背包问题介于01背包和完全背包之间，给出了每个物品被选择的次数 s。
比如输入为：N=2, C=5, v=\[1, 2\], w=\[1, 2\], s=\[2,1\]。数组s 表示v\[0\]可以选择2次，v\[1\]可以选择一次。

## 3.2 状态转移方程

```
dp[i][j] = max(dp[i-1][j], dp[i-1][j - k*v[i]] + k*w[i]), 0<k*v[i] <=j, 0<k<=s[i]
```

可以看出与完全背包一样，只是多了一个限制条件。
常规代码：

```
int maxValue(int N, int C, vector<int>& v, vector<int>& w, vector<int>& s){
    vector<vector<int>> dp(N, vector<int>(C+1));
    for(int j = 0; j <= C; j++){
        int maxK = min(j / v[0], s[j]);
        dp[0][j] = maxK * w[0];
    }
    
    for(int i = 1; i < N; i++){
        for(int j = 0; j <= C; j++){
            int no = dp[i-1][j];
            int yes = 0;
            for(int k = 1; k <= s[i]; k++){
                if(j < k*v[i]) break;
                yes = max(yes, dp[i-1][j - k*v[i]] + k*w[i]);
            }
            dp[i][j] = max(no, yes);
        }
    }
    return dp.back().back();
}
```

## 3.3 一维空间优化（无法降低时间复杂度）

在完全背包问题中只保留**容量维度**，并且可以切必须**从小到大**遍历容量，是因为每个物品可以选择无限次，不用考虑超出选择的次数。
而在多重背包问题中，必须要考虑物品的选择次数。因此多重背包的一维空间优化不能像完全背包一样降低时间复杂度。
代码

```
int maxValue(int N, int C, vector<int>& v, vector<int>& w, vector<int>& s){
    vector<int> dp(C+1);
    for(int i = 0; i < N; i++){
        for(int j = C; j >= 0; j--){
            for(int k = 0; k <= s[i] && j >= k* v[i]; k++){
              // 因为j是从C到0遍历，所以当k遍历时，始终只有dp[j]这个值在变化
              // j之前的dp是不变化的。
                dp[j] = max(dp[j], dp[j-k*v[i]] + k*w[i]); 
            }
        }
    }
    return dp.back();
}
```

## 3.4 二进制扁平化优化（可以降低时间复杂度）

**常规扁平化**转换，转换成**基本01背包问题**：把每件物品按照给定的次数复制，分别添加到体积数组v和价值数组w，更新后的v和w变为：

```
当s={2, 1, 3}时：
v = {v[0], v[0], v[1], v[2], v[2], v[2]}
w = {w[0], w[0], w[1], w[2], w[2], w[2]}
```

这样转换后就可以不考虑 s 数组，用基本01背包的方法进行解决。但是这种方法不会降低时间复杂度和空间复杂度。
### 二进制扁平化优化方法
优化的本质为一个十进制数可以用几个二进制数相加表示：
比如 $7=2^0 + 2^1 + 2^2$，当一个物品有7件时：
按照常规扁平化转换：

```
s={7}
v = {v[0], v[0], v[0], v[0], v[0], v[0], v[0]}
w = {w[0], w[0], w[0], w[0], w[0], w[0], w[0]}
```

按照二进制扁平化转换：

```
s={7}
v_bin = {v[0], 2*v[0], 4*v[0]}
w_bin = {w[0], 2*w[0], 4*w[0]}
```

相当于把7件同一个物品转换成了3个不同的商品，在原题中选择3件v\[0\]时，就对应为选择一件v\_bin\[0\]和一件v\_bin\[1\]，这样本来需要遍历3次，二进制转换后只需要遍历2次。
代码

```
int maxValue(int N, int C, vector<int>& v, vector<int>& w, vector<int>& s){
    vector<int> v_bin, w_bin;
    for(int i = 0; i < N; i++){
        int cnt = s[i];
        for(int j = 1; j <= cnt; j*=2){
            cnt--;
            v_bin.push_back(j * v[i]);
            w_bin.push_back(j * w[i]);
        }
        if(cnt>0){
            v_bin.push_back(cnt * v[i]);
            w_bin.push_back(cnt * w[i]);
        }
    }
    
    vector<int> dp(C+1);
    for(int i = 0; i < v_bin.size(); i++){
        for(int j = C; j >= v_bin[i]; j--){
            dp[j] = max(dp[j], dp[j - v_bin[i]] + w_bin[i]);
        }
    }
    return dp.back();
}
```

时间复杂度：$O(\sum_{i=0}^{N-1}log(s[i])*C)$
空间复杂度：$O(\sum_{i=0}^{N-1}log(s[i])*C)$

## 3.5 单调队列优化（进一步降低时间复杂度）
