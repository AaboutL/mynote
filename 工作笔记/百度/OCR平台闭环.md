# 回流
1. 数据如何方便使用，方便管理
	闭环要求工具和数据都放到同一个项目空间，数据按阶段分为原始数据、挖掘数据+mining_ref数据、标注数据、解析数据，命名规则。pdc上$AF_FLOW_YMD会自动对应当前的日期。
	| 类别 | 命名 |说明|
	| -- | -- | -- |
	| 原始数据         | origin-$垂类名-日期|日期格式：使用pdc自动生成：$AF_FLOW_YMD|
	| 挖掘数据         | mined-$垂类名-日期 |例如：mined-vat_invoice-20211124   |
	| mining_ref数据   | mining_ref-$垂类名-日期| 挖掘过程需要参考的难例 |
	| 标注数据         | labeled-$垂类名-日期|  |
	| 解析数据-主体     | parsed-zhuti-$垂类名-日期 |  |
	| 解析数据-主体分类  | parsed-cls-$垂类名-日期 |  |
	| 解析数据-四方向   | parsed-dir-$垂类名-日期 |  |
	| 解析数据-endet   | parsed-endet-$垂类名-日期 |  |
	| 解析数据-识别     | parsed-recog-$垂类名-日期 |  |
